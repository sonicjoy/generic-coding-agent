# Greet skill

When asked to add a greeting feature:

1. Create a module `greeting.py` exposing a `greet(name)` function.
2. `greet` should return the string `"Hello, {name}!"`.
3. Add a short docstring explaining the function.
4. Verify by importing and calling it.
